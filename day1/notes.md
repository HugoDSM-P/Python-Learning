# Day 1 – Python Basics

Today I started learning Python and covered the fundamental concepts of basic input and output.

## What I learned

* How to print text and values using `print()`
* How to format output using parameters like `sep`
* How to create line breaks with `\n`
* How to use escape characters like `\\` and `\t`
* How to take user input with `input()`
* How to combine strings using concatenation

## Summary

On my first day, I focused on understanding how Python interacts with the user through the console, including displaying information and receiving input.
