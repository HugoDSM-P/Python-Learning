# Day 9 – Modules, File Handling and Regular Expressions

Today I learned how to work with Python standard libraries, handle files, and use regular expressions to search for patterns in text.

## What I learned

### Modules

* Using built-in modules like:

  * `os` for file system navigation
  * `datetime` for working with dates and time
  * `time` for measuring execution time
  * `math` for mathematical operations
  * `collections` for advanced data structures

### Data Structures

* `Counter` to count occurrences
* `defaultdict` for default values in dictionaries
* `deque` for efficient list operations

### Regular Expressions

* Using `re` to search patterns in text
* Creating patterns with special characters
* Validating strings like emails and codes

### File Handling

* Opening and reading files
* Iterating through directories with `os.walk()`
* Processing multiple files automatically

## Project

I built a file scanner that:

* Searches through a directory and its subdirectories
* Opens `.txt` files and reads their content
* Uses regular expressions to find serial numbers with a specific pattern
* Stores and displays the results
* Measures execution time
* Shows formatted output with date and statistics

## Summary

This day introduced me to working with real-world data by combining file handling, pattern matching, and standard Python modules to automate tasks efficiently.
