# Day 10 – Game Development with Pygame

Today I built my first graphical game using Pygame, applying many Python concepts in a real-time environment.

## What I learned

### Pygame Basics

* Initializing a game with `pygame.init()`
* Creating a game window and setting title and icon
* Rendering images and text on screen
* Handling game loops and screen updates

### User Input

* Detecting keyboard events
* Handling key press (`KEYDOWN`) and release (`KEYUP`)
* Controlling player movement

### Game Logic

* Managing positions and movement of objects
* Implementing boundaries for the player
* Creating and controlling enemies
* Shooting mechanics and bullet state

### Collision Detection

* Calculating distance between objects
* Detecting collisions using math functions

### Sound and UI

* Adding background music with `mixer`
* Displaying score on screen
* Showing game over message

## Project

I developed a simple Space Invaders-style game that:

* Allows the player to move and shoot
* Spawns enemies with random positions
* Detects collisions between bullets and enemies
* Updates the score dynamically
* Ends the game when enemies reach a certain position

## Summary

This day helped me apply multiple programming concepts together in a real project, including game loops, real-time input, and graphical rendering.
