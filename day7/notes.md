# Day 7 – Object-Oriented Programming (OOP)

Today I learned the fundamentals of Object-Oriented Programming in Python and how to structure applications using classes and objects.

## What I learned

### Classes and Objects

* Defining classes with `class`
* Creating objects (instances)
* Using attributes and methods
* The `__init__` constructor

### Inheritance

* Creating child classes from parent classes
* Reusing and extending functionality with `super()`

### Special Methods

* `__str__` for readable object representation
* `__len__` to define object length
* `__del__` for object deletion behavior

### Other Concepts

* Class methods (`@classmethod`)
* Static methods (`@staticmethod`)
* Multiple inheritance
* Encapsulation of data and behavior

## Project

I built a simple banking system that:

* Creates a client with personal and account data
* Allows deposits and withdrawals
* Validates input and handles errors
* Displays client information
* Uses a menu-based interface for interaction

## Summary

This day introduced me to object-oriented programming, allowing me to build more structured and scalable programs by modeling real-world entities with classes.
