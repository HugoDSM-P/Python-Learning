# Day 4 – Random, Conditionals and Loops

Today I learned how to control the flow of a program using conditionals and loops, as well as how to generate random values.

## What I learned

### Random Module

* Generating random integers with `randint()`
* Generating random floats with `uniform()` and `random()`
* Selecting random elements with `choice()`
* Shuffling lists with `shuffle()`

### Conditionals

* Using `if`, `elif`, and `else`
* Nested conditionals
* Comparing values and using logical operators (`and`, `or`)

### Loops

* `for` loops to iterate over lists and ranges
* `while` loops for repeated execution
* Using `break` to stop loops
* Accumulating values inside loops

### Other Concepts

* Working with ranges
* Basic problem-solving with logic
* Boolean expressions

## Project

I built a number guessing game that:

* Generates a random number between 1 and 100
* Gives the user up to 8 attempts to guess it
* Provides hints (higher/lower)
* Validates input range
* Tracks the number of attempts

## Summary

This day helped me understand how to control program execution using logic and repetition, which are fundamental concepts in programming.
