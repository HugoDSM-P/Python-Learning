# Day 3 – Strings, Collections and Basic Logic

Today I explored more advanced string operations, data structures, and basic logical expressions in Python.

## What I learned

* String indexing and slicing
* Finding elements with `index()` and `rindex()`
* Reversing strings and stepping through them
* String methods like `join()` and `replace()`
* Repetition of strings using `*`

### Data Structures

* Lists: creation and methods like `append()`
* Dictionaries: accessing nested values
* Tuples: immutability, unpacking, and methods like `count()`
* Sets: uniqueness and methods like `update()`

### Other Concepts

* Type conversion between tuples and lists
* Boolean expressions and comparisons
* Membership operators (`in`)

## Project

I built a text analyzer that:

* Counts how many times specific letters appear in a text
* Calculates the length of the text
* Displays the first and last characters
* Reverses the text
* Checks if a specific word is present

## Summary

This day helped me understand how to manipulate strings and work with Python’s main data structures, as well as introducing basic logic and conditions.
