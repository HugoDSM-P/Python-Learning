# Day 2 – Data Types and Strings

Today I worked with core Python concepts related to data types and string handling.

## What I learned

* How to create and use variables
* Basic data types: `int`, `float`, and `str`
* How to check types using `type()`
* Type casting between numbers and strings
* Basic arithmetic operations
* String concatenation
* Formatted strings using f-strings
* Rounding numbers with `round()`

## Project

I built a simple program that:

* Asks the user for their name and sales
* Calculates a 13% commission
* Displays the result formatted to 2 decimal places

## Summary

This day helped me understand how Python handles different types of data and how to format and manipulate output effectively.
