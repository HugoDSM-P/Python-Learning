# Day 5 – Functions and Code Organization

Today I learned how to structure programs using functions and how to write more reusable and organized code.

## What I learned

### Functions

* Defining functions with `def`
* Using parameters and return values
* Difference between `print()` and `return`
* Reusing code with functions

### Arguments

* Positional arguments
* Arbitrary arguments with `*args`
* Keyword arguments with `**kwargs`

### Logic and Problem Solving

* Iterating through lists inside functions
* Conditional logic inside functions
* Breaking down problems into smaller parts

## Project

I built a simple Hangman game that:

* Selects a random word
* Displays hidden letters using underscores
* Lets the user guess letters
* Tracks used letters and remaining attempts
* Updates the display as the user guesses correctly
* Ends when the word is guessed or attempts run out

## Summary

This day helped me understand how to organize code using functions, making programs more readable, reusable, and closer to real-world applications.
